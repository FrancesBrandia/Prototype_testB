Welcome to the ACR's Master Component Library!

The purpose of this library is to provide ACR developers with a foundation for some of the most common components used in ACR web applications. This will help to maintain consistent styling across applications and significantly reduce the time needed to develop them.

How to use the library

Explore the various sections of components via the sidebar to the left. Underneath each component is the associated HTML (and JavaScript, if applicable) needed to reconstruct the component.

If you find a component you need, copy its HTML, CSS and JavaScript and use it in your application. Edit the content and/or styling of the component to suit the needs of your app. The main structure, padding and alignment of the component will remain intact.

How to access the CSS for the components

Each page of components has a title at the top. Click on the title to be directed to the CSS file which contains the styling for all the components within that page.

For example, the title on the Action Buttons page is a link to the CSS file that has the styling for all the action buttons displayed on the page.

Global styles

CSS

All of the components in this library have some default styles applied to them.

Make sure to copy the CSS below into your application to make sure the components behave properly:

* {
    margin: 0;
    border: 0;
    padding: 0;
    box-sizing: border-box;
    color: #444444;
}

/* prevents users from accidentally highlighting icons when clicking them */
.material-icons {
    user-select: none;
}

Font

The main font used in this library is Roboto from Google Fonts.

Be sure to include the following code in the <head> of your HTML file:

<link rel="preconnect" href="https://fonts.gstatic.com"> 
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto&display=swap">

You can apply the font by using the CSS below:

body {
    font-family: "Roboto", sans-serif;
}

Icons

The icons used in this library come from Google's Material Icons library.

In order for these icons to work, you must include the following code in the head of your HTML file:

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Click here to view all of the Material Icons and to see more detailed instructions on how to use them.

Disclaimers

Responsiveness

These components are designed primarily for tablet and desktop applications.

Repetitive code

Some components utilize the same CSS/JavaScript from other comonents.

For example, there is a dropdown single select component in Elements, Inputs and Controls. This same dropdown is used within several other components, meaning the same CSS and JavaScript for the dropdown is also included in those other components.

CSS classes

Some of the CSS classes may override with other components from this library or from your application. Keep this in mind when using these components and if you encounter any unexpected behavior.

If you change the class names in a component, make sure to update the CSS file and the JavaScript (if applicable) for the component accordingly.
